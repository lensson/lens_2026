Description:
This sample creates a multicast user port.

Prerequisite:
device has to be created.
ONU ANI has to be created.

Input parameters:

* cc-name: Cross Connect Vlan sub Interface
* multicast-package-name: The name of a multicast package
* onu-name: ONU name
* user-port-name: Multicast VPN User Port Name
* vpn-name: VPN Name
