Description:
This sample configures the VOIP DUAL SIP CLIENT interfaces.

Prerequisite:
VONU device has to be created.


Input parameters:

* call-statistics-enabled: Enable the VoIP call statistics for a particular POTS line.
* onu-name: ONU name
* pots-port-hw-number: ONU POTS UNI Number
* sip-admin: Valid values, [true, false]
* veip-interface-name: VEIP interface which is used to manage VoIP services
* voip-uni-name: VOIP UNI name
