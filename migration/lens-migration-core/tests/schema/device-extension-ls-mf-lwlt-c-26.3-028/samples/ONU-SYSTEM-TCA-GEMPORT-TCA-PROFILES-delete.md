Description:
This macro removes the TCA profile for GEM port performance monitoring counters in an ONU UNI interface (ENET)

Prerequisite:

* GEM port PM TCA profile has to be created

Input parameters:

* gemport-tca-profile    :  gemport-tca-profile name

Input parameters:

* gemport-tca-profile: GEM port performance monitoring TCA threshold profile name
* onu-name: ONU name
