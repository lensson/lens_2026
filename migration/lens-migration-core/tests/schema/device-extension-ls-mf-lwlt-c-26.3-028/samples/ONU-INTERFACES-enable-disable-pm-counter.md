This macro enables or disables the PM counters for an ANI interface (FEC and BIP) on an ONU.

Prerequisite:

*   device has to be created.
*   ANI interface has to be created.

Input parameters:

* onu-name: ONU name
* onu-template-ani-intf: ANI interface name
* pm-flag: **true** to enable or **false** to disable PM counters for an ANI
