This macro config pbit marking list to tc. 

Pre-condition: 

 * Device has to be created.


Input parameters:

* onu-name: ONU name
* pbit0-tc: Traffic Class Value for PBIT 0
* pbit1-tc: Traffic Class Value for PBIT 1
* pbit2-tc: Traffic Class Value for PBIT 2
* pbit3-tc: Traffic Class Value for PBIT 3
* pbit4-tc: Traffic Class Value for PBIT 4 (if specified)
* pbit5-tc: Traffic Class Value for PBIT 5 (if specified)
* pbit6-tc: Traffic Class Value for PBIT 6 (if specified)
* pbit7-tc: Traffic Class Value for PBIT 7 (if specified)
* qos-policy-name: Policy name
