Description:
This sample provide PING test.

Prerequisite:
VONU device has to be created.
IP host has to be created.

Input parameters:

* count: the number of the PING attempts
* onu-name: ONU name
* ping-test-name: PING test instance name
* target-domain-name: target domain name of the PING
* target-ip-address: target IP address of the PING
