This macro removes all the IPv6 addresses learned dynamically from DHCPv6 on the VLAN sub-interface.

Prerequisite:

* learn-addresses-from-dhcpv6 must be enabled on the VLAN sub-interface.

Input parameters:

* cc-name: The name of CC sample. In IBRIDGE senario, leave it empty
* ibridge-user-port-name: The name of IBRIDGE-USER-PORT sample. In CC senario, leave it empty
