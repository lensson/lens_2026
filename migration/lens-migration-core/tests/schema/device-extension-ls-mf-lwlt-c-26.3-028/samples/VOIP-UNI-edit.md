This macro configures the VOIP uni instance.

Prerequisite:
ONU device has to be created.

Input parameters:

* call-statistics-enabled: Enable the VoIP call statistics for a particular POTS line.
* onu-name: ONU name
* onu-template-voip-uni-name: ONU template VoIP UNI interface name
* sip-display-name: This ASCII string attribute defines the customer ID used for the display attribute in outgoing SIP messages.
* sip-password: The SIP user name and password are used for authentication
* sip-user-aor: The user identification part of the address of record
* sip-user-name: The SIP user name and password are used for authentication
