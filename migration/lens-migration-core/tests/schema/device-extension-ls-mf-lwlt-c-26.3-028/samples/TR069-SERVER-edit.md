This macro configures the TR069 SERVER instance.

Prerequisite:
ONU device has to be created.

Input parameters:

* admin-state: In almost all the deployment scenarios, the TR-069 connection is configured with unlocked.
* authentication-user-name: Authentication user name
* authentication-user-password: authentication user password.
* onu-name: ONU name
* onu-template-name: ONU template name
* onu-template-VEIP: Identifies within the ONU template the VEIP interface that is to be instantiated within the actual ONU instance.The actual attributes will have the data as configured in the template but completed / overruled with the data provided in this list entry.
