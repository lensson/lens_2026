This macro removes all the IPv4 addresses learned dynamically from DHCP on the VLAN sub-interface.

Prerequisite:

* learn-addresses-from-dhcp must be enabled on the VLAN sub-interface.

Input parameters:

* cc-name: The name of CC sample. In IBRIDGE senario, leave it empty
* ibridge-user-port-name: The name of IBRIDGE-USER-PORT sample. In CC senario, leave it empty
