This macro creates the UNICAST SERVICE QoS policy profile.

Prerequisite: OLT device has to be created.

Input parameters:

* ccl-policy-name: CCL policy name
* color-policy-name: color policy name
* count-policy-name: count policy name
* dscp-to-pbit-alignment-policy-name: DSCP to Pbit policy name
* igmp-remarking-policy-name: Remark IGMP policy name
* pbit-scheduling-policy-name: Scheduling policy name
* policing-policy-name: Policing policy name
* policing-tc-policy-name: Policing tc policy name
* policy-profile-name: QoS policy profile name
* remarking-policy-name: Remark policy name
* untag-policy-name: Untag policy name
