This macro modifies the set of N:1-VLAN Forwarder Profiles parameters.

Input parameters:

* in-port: The interface usage type of the in port
* mac-can-not-move-to: The interface-usage value to which MAC addresses can not move
* out-port: The interface usage type of the out port
* receive-interface-usage: Defines MAC address learning rules for frames received on a port for which the interface usage of the underlying interface is of this value
