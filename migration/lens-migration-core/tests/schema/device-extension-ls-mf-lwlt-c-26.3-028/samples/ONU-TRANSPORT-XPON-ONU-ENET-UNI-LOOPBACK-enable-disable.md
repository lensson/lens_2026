Description:
This macro enable or disable loopback of an UNI interface on an ONU.

Prerequisite:
*   device has to be created.
*   UNI interface has to be created and up.

Input parameters:

* enet-name: UNI interface name
* loopback-duration: Indicates for how long the interface shall be kept in loopback mode, Values Range:[infinite,1-93600seconds]
* loopback-flag: enable or disable UNI loopback on an ONU
* onu-name: ONU name
