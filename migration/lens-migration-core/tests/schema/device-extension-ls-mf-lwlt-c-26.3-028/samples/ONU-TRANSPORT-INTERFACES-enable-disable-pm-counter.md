Description:
This sample enables or disables the PM counters for an ANI interface on an ONU.

Prerequisite:
*   device has to be created.
*   ANI interface has to be created.

Input parameters:

* onu-name: ONU name
* onu_ani_intf: ANI interface name
* pm_flag: **true** to enable or **false** to disable PM counters for an ANI
