Description:
This sample modifies the TC 2 Q profile.

Prerequisite:
device has to be created.

Input parameters:

* onu-name: ONU name
* tc-2-q-profile-name: TC 2 Queue Id Mapping
* tc0-queue-id: Traffic Class Id mapped to Queue 0
* tc1-queue-id: Traffic Class Id mapped to Queue 1
* tc2-queue-id: Traffic Class Id mapped to Queue 2
* tc3-queue-id: Traffic Class Id mapped to Queue 3
* tc4-queue-id: Traffic Class Id mapped to Queue 4 (empty unless specified)
* tc5-queue-id: Traffic Class Id mapped to Queue 5 (empty unless specified)
* tc6-queue-id: Traffic Class Id mapped to Queue 6 (empty unless specified)
* tc7-queue-id: Traffic Class Id mapped to Queue 7 (empty unless specified)
