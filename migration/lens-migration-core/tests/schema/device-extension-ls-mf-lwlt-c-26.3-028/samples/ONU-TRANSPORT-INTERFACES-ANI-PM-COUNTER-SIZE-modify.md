Description:
This macro modifies pm counter size of an ANI interface on an ONU.
Prerequisite:
*   device has to be created.
*   ANI interface has to be created.
Input parameters:

* onu-name: ONU name
* onu_ani_intf: ANI interface name
* pm_counter_size: Modifies pm counter size of an ANI interface,Values Range:[32bit-performance-monitoring,64bit-performance-monitoring]
