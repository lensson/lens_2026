This macro configures SFP transceiver-link TCA profile for rx-power, tx-bias and tx-power.

Pre-condition: 

 * SFP transceiver-link-tca-profile has to be created.

Input parameters:

* rx-power-threshold-high-alarm: Value for rx-power-threshold-high-alarm range is -1270 .. 200
* rx-power-threshold-high-warning: Value for rx-power-threshold-high-warning range is -1270 .. 200
* rx-power-threshold-low-alarm: Value for rx-power-threshold-low-alarm range is -1270 .. 200
* rx-power-threshold-low-warning: Value for rx-power-threshold-low-warning range is -1270 .. 200
* transceiver-link-tca-profile-name: transceiver link tca profile name
* tx-bias-threshold-high-alarm: Value for tx-bias-threshold-high-alarm range is 0 .. 250000
* tx-bias-threshold-high-warning: Value for tx-bias-threshold-high-warning range is 0 .. 250000
* tx-bias-threshold-low-alarm: Value for tx-bias-threshold-low-alarm range is 0 .. 250000
* tx-bias-threshold-low-warning: Value for tx-bias-threshold-low-warning range is 0 .. 250000
* tx-power-threshold-high-alarm: tx-power-threshold-high-alarm range is -1270 .. 200
* tx-power-threshold-high-warning: tx-power-threshold-high-alarm range is -1270 .. 200
* tx-power-threshold-low-alarm: Value for tx-power-threshold-low-alarm range is -1270 .. 200
* tx-power-threshold-low-warning: Value for tx-power-threshold-low-warning range is -1270 .. 200


Input parameters:

* onu-name: ONU name
* rx-power-threshold-high-alarm: Value for rx-power-threshold-high-alarm range is -1270 .. 200
* rx-power-threshold-high-warning: Value for rx-power-threshold-high-warning range is -1270 .. 200
* rx-power-threshold-low-alarm: Value for rx-power-threshold-low-alarm range is -1270 .. 200
* rx-power-threshold-low-warning: Value for rx-power-threshold-low-warning range is -1270 .. 200
* transceiver-link-tca-profile-name: transceiver link tca profile name
* tx-bias-threshold-high-alarm: Value for tx-bias-threshold-high-alarm range is 0 .. 250000
* tx-bias-threshold-high-warning: Value for tx-bias-threshold-high-warning range is 0 .. 250000
* tx-bias-threshold-low-alarm: Value for tx-bias-threshold-low-alarm range is 0 .. 250000
* tx-bias-threshold-low-warning: Value for tx-bias-threshold-low-warning range is 0 .. 250000
* tx-power-threshold-high-alarm: tx-power-threshold-high-alarm range is -1270 .. 200
* tx-power-threshold-high-warning: tx-power-threshold-high-alarm range is -1270 .. 200
