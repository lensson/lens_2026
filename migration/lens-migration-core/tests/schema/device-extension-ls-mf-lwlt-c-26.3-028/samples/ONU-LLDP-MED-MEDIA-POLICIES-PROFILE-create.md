Description:
This macro creates the LLDP-MED network media policy profile to be associated to the LLDP port. The profile can be used to define the network media policy for 9 different pre-defined application types in ONU.

Prerequisite:
None.
Input parameters:

* dscp-streaming-video-app-all-apps-profile: DSCP in the media policy configured for streaming-video application type in policy profile.
* guest-voice-app-type: Application type in the media end point device is guest-voice.
* guest-voice-signaling-app-type: Application type in the media end point device is guest-voice-signalling.
* lldp-med-media-policies-for-all-apps: LLDP MED media policy profile name which defines policies for all application types.
* lldp-med-plug-and-play-lan-policy-for-voice: LLDP MED media policy profile name for VOIP plug-and-play.
* onu-name: ONU name
* pri-guest-voice-app-all-apps-profile: Priority bits in the media policy configured for guest-voice application type in policy profile.
* pri-guest-voice-signaling-app-all-apps-profile: Priority bits in the media policy configured for guest-voice-signaling application type in policy profile.
* pri-softphone-voice-app-all-apps-profile: Priority bits in the media policy configured for softphone-voice application type in policy profile.
* pri-video-conferencing-app-all-apps-profile: Priority bits in the media policy configured for video-conferencing application type in policy profile.
* pri-video-signaling-app-all-apps-profile: Priority bits in the media policy configured for video-signaling application type in policy profile.
* pri-voice-app-all-apps-profile: Priority bits in the media policy configured for voice application type in policy profile.
* pri-voice-app-voice-profile: Priority bits in the media policy configured for voice application type in voice policy profile.
* pri-voice-signaling-app-all-apps-profile: Priority bits in the media policy configured for voice-signaling application type in policy profile.
* softphone-voice-app-type: Application type in the media end point device is softphone.
* streaming-video-app-type: Application type in the media end point device is streaming-video.
* unknown-app-type: Application type in the media end point device is unknown.
* video-conferencing-app-type: Application type in the media end point device is vide-conferencing.
* video-signaling-app-type: Application type in the media end point device is video-signaling.
* vlan-id-guest-voice-app-all-apps-profile: VLAN ID in the media policy configured for guest-voice application type in policy profile.
* vlan-id-guest-voice-signaling-app-all-apps-profile: VLAN ID in the media policy configured for guest-voice-signaling application type in policy profile.
* vlan-id-softphone-voice-app-all-apps-profile: VLAN ID in the media policy configured for softphone-voice application type in policy profile.
* vlan-id-video-conferencing-app-all-apps-profile: VLAN ID in the media policy configured for video-conferencing application type in policy profile.
* vlan-id-video-signaling-app-all-apps-profile: VLAN ID in the media policy configured for video-signaling application type in policy profile.
* vlan-id-voice-app-all-apps-profile: VLAN ID in the media policy configured for voice application type in policy profile.
* vlan-id-voice-app-voice-profile: VLAN ID in the media policy configured for voice application type in voice policy profile.
* vlan-id-voice-signaling-app-all-apps-profile: VLAN ID in the media policy configured for voice-signaling application type in policy profile.
* voice-app-type: Application type in the media end point device is voice.
* voice-signaling-app-type: Application type in the media end point device is voice-signaling.
