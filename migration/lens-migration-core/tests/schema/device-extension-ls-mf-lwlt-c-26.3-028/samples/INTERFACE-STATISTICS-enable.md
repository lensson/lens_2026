This macro enables interface statistics.

Pre-condition:

* VENET-INTERFACE-NAME

* There are best-effort/false 2 modes.
*	best-effort: This is the default option, under this, in/out bytes, in/out discard packets , in/out pkts are supported and collected;
*	false: under this option, statistics are not collected.

Input parameters:

* interface-statistics-mode: 
* venet-interface-name: the name of the olt-v-enet who will enable interface statistics.
