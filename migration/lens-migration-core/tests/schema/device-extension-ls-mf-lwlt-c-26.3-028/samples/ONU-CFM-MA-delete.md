This macro deletes Connectivity Fault Management (CFM) MA template parameters in a network 

Prerequisite:

*   device has to be created.
*   md-id and ma-id should be initially configured, and are the keys for  maintenance domain and maintenance association respectively.
Input parameters:

* ma-id: Unique ID for MA.
* md-id: Unique ID which is a key for maintenance domain
* onu-name: ONU name
