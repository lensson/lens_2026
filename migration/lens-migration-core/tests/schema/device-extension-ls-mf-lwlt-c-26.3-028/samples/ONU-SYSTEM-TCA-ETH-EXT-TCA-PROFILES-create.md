Description:
This macro configures TCA profiles for Ethernet frame extended performance monitoring counters on an ONT UNI interface (ENET)

Prerequisite:
* None

Input parameters:
* in-drop-frames-threshold    :  Threshold value for the total number of inbound frames dropped range is 1 .. 4294967294
* out-drop-frames-threshold    :  Threshold value for the total number of outbound frames dropped range is 1 .. 4294967294
* in-drop-octets-threshold :  Threshold value for the total number of inbound bytes dropped range is 1 .. 4294967294


Input parameters:

* ethernet-frame-error-extended-profile: Ethernet frame performance monitoring counters threshold profile name
* in-drop-frames-threshold: Threshold value for the total number of inbound frames dropped range is 0 .. 4294967294
* in-drop-octets-threshold: Threshold value for the total number of inbound bytes dropped range is 0 .. 4294967294
* onu-name: ONU name
* out-drop-frames-threshold: Threshold value for the total number of outbound frames dropped range is 0 .. 4294967294
