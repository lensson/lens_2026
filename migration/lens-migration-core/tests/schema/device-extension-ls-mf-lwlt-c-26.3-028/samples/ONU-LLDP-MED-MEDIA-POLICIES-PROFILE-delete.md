Description:
This macro deletes the LLDP-MED network media policy profile. 

Prerequisite:
- LLDP-MED media policy profile to be deleted is already created. 
- LLDP-MED media policy profile to be deleted is not associated to the LLDP port.
Input parameters:

* lldp-med-media-policies-for-all-apps: LLDP MED media policy profile name which defines policies for all application types.
* lldp-med-plug-and-play-lan-policy-for-voice: LLDP MED media policy profile name for VOIP plug-and-play.
* onu-name: ONU name
