Description:
This macro configures TCA profiles for the CPU utilization of a CPU component.

Prerequisite:
 CPU component has to be created.

Input parameters:
cpu-threshold    :  Value for CPU threshold range is 0..100

Input parameters:

* cpu-tca-profile: CPU threshold profile name
* cpu-tca-threshold: value for threshold range is 0..100
* onu-name: ONU name
