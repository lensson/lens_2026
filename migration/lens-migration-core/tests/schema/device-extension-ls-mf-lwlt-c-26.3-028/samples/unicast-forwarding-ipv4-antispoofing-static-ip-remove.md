This macro removes the IPv4 static address configured on the VLAN sub-interface.

Prerequisite:

* Static IPv4 address must be configured on the VLAN sub-interface.

Input parameters:

* cc-name: The name of CC sample. In IBRIDGE senario, leave it empty
* ibridge-user-port-name: The name of IBRIDGE-USER-PORT sample. In CC senario, leave it empty
* ip-address: An IPv4 address assigned to the interface
* prefix-length: The length of the subnet prefix
