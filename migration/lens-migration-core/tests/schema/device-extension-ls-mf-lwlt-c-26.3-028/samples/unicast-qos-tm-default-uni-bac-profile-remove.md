This macro deletes the default set of QoS Traffic Management(TM) Buffer 
Acceptance Control(BAC) profiles for user interfaces.

The result is the deletion of the following set of profiles:

1. BAC profile with the name 'UNI_BECLQ'.
2. BAC profile with the name 'UNI_VideoQ'.
3. BAC profile with the name 'UNI_VoiceQ'.

Prerequisite: 

 No prerequisites.
