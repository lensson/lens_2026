This macro allows to enable or disable the u2u-loop-avoidance-check to exclude the user interface where the traffic is originated and looped back to same user interface, from downstream flooding traffic including broadcast, unknown unicast and unknown multicast.

Prerequisites:

No prerequisites.

Input parameters:

* u2u-loop-avoidance-check-enable: Setting this parameter to 'true' will enable the u2u-loop-avoidance-check to exclude the user interface where the traffic is originated and looped back to same user interface, from downstream flooding traffic including broadcast, unknown unicast and unknown multicast.
