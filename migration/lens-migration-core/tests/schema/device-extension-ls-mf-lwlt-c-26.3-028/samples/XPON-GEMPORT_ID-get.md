This macro generate the GEM port ID parameters.

Prerequisite:

*The referenced Channel group must exist in OLT side.

Input parameters:

* allowed-gemport-id-list: An allowed ID list for allocating free GEM port IDs, the format can be 257-1915 or 257,260,270 or 256-300,400-1915
* cc-name: CC name
* ibridge-user-port-name: Ibridge user port name
* olt-port-name: Channel group
* xpon-technology: Type of Pon
