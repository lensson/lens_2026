Description:
This macro removes TCA profiles for Ethernet frame extended performance monitoring counters on an ONT UNI interface (ENET)

Prerequisite:
* Ethernet frame extended performance monitoring TCA profile has to be removed

Input parameters:
* ethernet-frame-error-extended-profile    :  ethernet-frame-error-extended-profile name


Input parameters:

* ethernet-frame-error-extended-profile: Ethernet frame performance monitoring counters threshold profile name
* onu-name: ONU name
