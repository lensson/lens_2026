Description:
This sample instantiates ONU instance multicast vsi inteface

Prerequisite:
*   ONU instance has to be created.

Input parameters:

* admin-state: enable or disable mcast vsi in onu instance
* mcast-vlan-id: the vlan id for multicast
* onu-name: ONU name
* onu-template-mc-vsi-name: multicast vsi name from onu template
