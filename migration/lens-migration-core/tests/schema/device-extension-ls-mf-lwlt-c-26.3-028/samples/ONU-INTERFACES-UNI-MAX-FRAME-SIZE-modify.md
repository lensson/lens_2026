This sample defines or modifies maximum frame size that can be allowed across the Ethernet interface.

Prerequisite:

*   Device has to be created
*   ENET UNI interface has to be created


Input parameters:

* enet-frame-size: Ethernet max frame size to be allowed across the ENET interface
* onu-name: ONU name
* onu-template-uni-intf: ENET interface name
