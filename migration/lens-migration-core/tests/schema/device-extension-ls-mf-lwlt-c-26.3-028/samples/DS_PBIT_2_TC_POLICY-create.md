This macro creates the UNICAST SERVICES QoS PBIT 2 TC policy.

Prerequisite: OLT device has to be created.

Input parameters:

* ds-pbit-2-tc-policy-name: Policy name
* pbit0-tc: Traffic class to be associated with pbit0
* pbit1-tc: Traffic class to be associated with pbit1
* pbit2-tc: Traffic class to be associated with pbit2
* pbit3-tc: Traffic class to be associated with pbit3
* pbit4-tc: Traffic class to be associated with pbit4
* pbit5-tc: Traffic class to be associated with pbit5
* pbit6-tc: Traffic class to be associated with pbit6
* pbit7-tc: Traffic class to be associated with pbit7
* vlan-tag: Vlan tag index
