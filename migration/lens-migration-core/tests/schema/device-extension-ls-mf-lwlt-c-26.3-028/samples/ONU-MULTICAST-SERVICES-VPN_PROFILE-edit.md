Description:
This sample configures a multicast vpn profile.

Prerequisite:
device has to be created.
ONU ANI has to be created.

*	{IGMP-PROTOCOL-VERSION} , IGMP Protocol Version - Accepted Values [2,3]

Input parameters:

* igmp-protocol-version: IGMP protocol version value
* mc-profile-name: The name of the multicast profile.
* onu-name: ONU name
