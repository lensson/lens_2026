This macro gets the Connectivity Fault Management (CFM) stack for all/specific ports:

- Configuring **port**, which is an interface that represents the Port on which MEPs or MIP might be configured

Prerequisite:
CFM parameters has to be created.

Input parameters:

* onu-name: ONU name
* port: An interface that represents the Bridge Port or Aggregation Port on which MEPs or MIPs might be configured
