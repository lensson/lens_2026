This macro edits a background rogue test on the channel termination.

Prerequisite:
olt infrastructure created

Input parameters:

* activate: Enable or disable background rogue test.
* bkg-test-name: Name of the background rogue test.
* olt-infrastructure-name: The name of the OLT infrastructure on which channel termination created.
* rogue-test-profile-ref: Reference to a rogue test profile.
