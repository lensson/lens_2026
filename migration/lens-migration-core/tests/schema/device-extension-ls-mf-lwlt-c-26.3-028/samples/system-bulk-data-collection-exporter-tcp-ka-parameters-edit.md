This macro configures the TCP Keep-Alive parameters for the connection to the IPFIX Collector. If the parameters are not configured the default TCP Keep-Alive (i.e., idle time 7200secs) is applied.

Input parameters:

* exporter-name: exporter name
* idle-time: idle time (in secs)
* max-probes: max probes
* probe-interval: probe interval (in secs)
