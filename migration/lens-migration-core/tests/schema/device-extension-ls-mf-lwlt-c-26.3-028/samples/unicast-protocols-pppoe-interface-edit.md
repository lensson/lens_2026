This macro attaches the PPPoE Intermediate Agent profile to VLAN sub-interface.

Prerequisite:

* PPPoE Intermediate Agent profile should exist

* Service should exist

Input parameters:

* cc-name: CC user port name
* enable: Enable PPPoE Intermediate Agent when configured as true
* ibridge-user-port-name: Ibridge user port name
* pppoe-profile-name: Name of the PPPoE Intermediate Agent profile reference
