This macro retrieves the operational data of IPFIX exporter(s) on LightSpan.
In summary, it reports
  - the connectivity status/info of each of the exporter's destinations.
  - start and stop timing information of each of the exporter's destinations.
  - the number of templates and data records of each of the exporter's destinations.
  - details on the data records that each of the exporter's destinations have exported.

Note:
  The retrieved data also includes both operational and configuration data, given that the model is mixed
