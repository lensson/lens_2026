Description:
This sample enables or disables the PM counters for an MoCA uni interface on an ONU.

Prerequisite:
*   device has to be created.
*   MoCA Uni interface has to be created.
Input parameters:

* moca-name: MoCA uni interface name
* onu-name: ONU name
* pm-flag: **true** to enable or **false** to disable PM counters for an MoCA UNI interface
