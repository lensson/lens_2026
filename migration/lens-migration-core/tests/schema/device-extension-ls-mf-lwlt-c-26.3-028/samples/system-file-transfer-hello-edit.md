This macro configures the preferred TLS version or a set of ciphers that will be used during the handshake phase. The default TLS versions and ciphers, as well as their preference order, 
can be found in appendix section _Supported TLS versions and ciphers_.

Input parameters:

* cipher-suite: YANG identifier of a TLS cipher to add
* tls-version: YANG identifier of a TLS protocol version to add
