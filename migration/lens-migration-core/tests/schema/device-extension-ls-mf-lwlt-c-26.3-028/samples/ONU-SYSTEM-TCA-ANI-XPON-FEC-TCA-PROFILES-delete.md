Description:
This macro removes the ANI XPON FEC performance monitoring TCA profile for an ONT ANI interface

Prerequisite:

* ANI XGPON FEC PM TCA profile has to be created

Input parameters:

* xgpon-fec-layer-tca-profile    :  xgpon-fec-layer-tca-profile name

Input parameters:

* onu-name: ONU name
* xpon-fec-error-profile: ANI XPON FEC PM counters threshold profile name
