Description:
This sample configures the MoCA interfaces.

Prerequisite:
device has to be created.
Input parameters:

* moca-mode: configure ONU support MoCA1.1 or MoCA2.5
* moca-name: The name of the ONU MoCA interface that will be created
* moca-password: MoCA Encryption Key
* moca-port-hw-number: ONU MoCA uni Number
* moca-port-hw-option: ONU MoCA HW Option
* onu-name: ONU name
