This macro edits a 25GSPON port infrastructure in the OLT which can be used as underlying hardware for a channel-termination interface in a 25GSPON OLT infrastructure. The 25GSPON port infrastructure consists of one transceiver-link-ngpon and one channel-group inside a transceiver cage located on a board. This macro can only be used for 25GSPON type.

Input parameters:

* model-name: Model number of SFP plugged in the cage (model number can be a generic model-name or a dedicated 3FE number).
* olt-port-name: OLT infrastructure port name.
* xpon-cage-name: Cage name.
