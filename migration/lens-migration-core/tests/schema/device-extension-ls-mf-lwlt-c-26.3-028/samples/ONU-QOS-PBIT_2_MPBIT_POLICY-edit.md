Description:
This sample configures the priority bit to marked pbit parameters.

Prerequisite:
device has to be created.
Ethernet port has to be created

Input parameters:

* onu-name: ONU name
* pbit-2-mpbit-policy: Pbit To Marked Pbit
* pbit0-mpbit: Value for marked pbit to traffic class 0
* pbit1-mpbit: Value for marked pbit to traffic class 1
* pbit2-mpbit: Value for marked pbit to traffic class 2
* pbit3-mpbit: Value for marked pbit to traffic class 3
* pbit4-mpbit: Value for marked pbit to traffic class 4 (if specified)
* pbit5-mpbit: Value for marked pbit to traffic class 5 (if specified)
* pbit6-mpbit: Value for marked pbit to traffic class 6 (if specified)
* pbit7-mpbit: Value for marked pbit to traffic class 7 (if specified)
