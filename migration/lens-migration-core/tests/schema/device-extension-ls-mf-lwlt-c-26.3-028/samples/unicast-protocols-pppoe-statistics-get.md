This macro retrieves PPPoE Intermediate Agent statistics from a VLAN sub-interface.

Prerequisite:

* PPPoE Intermediate Agent must be enabled on user VLAN sub-interface.

Note : Reset of VLAN sub-interface statistics resets PPPoE Intermediate Agent statistics too

Input parameters:

* cc-name: CC user port name for PPPoE Intermediate Agent statistics counter needs to be retrived
* ibridge-user-port-name: Ibridge user port name for PPPoE Intermediate Agent statistics counter needs to be retrived
