This macro creates a policing pre-handling profile.

* create a policing pre-handling profile


Input parameters:

* dei0-color: pre-color for dei 0
* dei1-color: pre-color for dei 1
* pbits-green: pbits that pre-colored as green
* pbits-red: pbits that pre-colored as red
* pbits-yellow: pbits that pre-colored as yellow
* policing-pre-handling-profile-name: policing-pre-handling-profile name
* vlan-tag: Vlan tag index
