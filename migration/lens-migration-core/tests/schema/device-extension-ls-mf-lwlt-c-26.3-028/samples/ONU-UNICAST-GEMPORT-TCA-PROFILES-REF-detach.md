Description:
This macro detaches TCA profile for GEM port performance monitoring counters from an ONU GEM port

Prerequisite:

* GEM port TCA profile should be already associated with gemport-tca-profile

Input parameters:

* gemport-tca-profile    :  gemport-tca-profile name

Input parameters:

* gemport-name: GEM port name for which TCA should be enabled or disabled
* gemport-tca-profile: GEM port PM TCA profile name
* onu-name: ONU name
