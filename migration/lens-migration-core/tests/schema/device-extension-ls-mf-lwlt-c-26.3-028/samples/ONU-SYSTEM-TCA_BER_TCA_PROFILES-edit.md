This macro configures BER TCA profile for sd-threshold and sf-threshold.

Pre-condition:

 * None

Input parameters:

* sd-threshold: Value for sd-threshold range is 6 .. 9
* sf-threshold: Value for sf-threshold range is 5 .. 8


Input parameters:

* ber-tca-profile: BER TCA Profile Name
* onu-name: ONU name
* sd-threshold: Value for SD Threshold range is 6 .. 9
* sf-threshold: Value for SF Threshold range is 5 .. 8
