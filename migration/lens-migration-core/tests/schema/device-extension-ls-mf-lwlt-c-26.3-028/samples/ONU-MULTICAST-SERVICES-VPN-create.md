Description:
This sample creates a multicast vpn.

Prerequisite:
device has to be created.
ONU ANI has to be created.
Multicast interface has to be created.
Multicast profile has to be created.

Input parameters:

* mc-profile-name: The name of the multicast profile.
* onu-name: ONU name
* vpn-name: The name of a multicast vpn.
