This macro retrieves the QoS Traffic Management(TM) Buffer Acceptance 
Control(BAC) profile configuration.

Prerequisite: 

 No prerequisites.

Input parameters:

* bac-prof-name: Name of the Buffer Acceptance Control(BAC) profile. If this is not provided, then all the configured BAC profiles will be retrieved.
* detailed-output: Provides detailed configuration data of all the BAC profiles in the system when enabled and only BAC profile names when disabled. This argument will be ignored when bac-prof-name is provided.
