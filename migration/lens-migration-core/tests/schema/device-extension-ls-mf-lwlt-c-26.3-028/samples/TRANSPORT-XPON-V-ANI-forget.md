Allows to clean up the results produced by the defective ONU when the operator confirmed it
has been dealt with.

Pre-condition:

* The defective ONU ploam-condition got disabled by the operator. 
* The defective ONU got disconnected or repaired and re-enabled.

Input parameters:

* defect: The specific ONU defect for which the handling procedure has been completed
* olt-infrastructure-name: OLT infrastructure channel termination name.
* onu-serial-number: Serial number of the defective ONU for which the handling procedure has been completed.
