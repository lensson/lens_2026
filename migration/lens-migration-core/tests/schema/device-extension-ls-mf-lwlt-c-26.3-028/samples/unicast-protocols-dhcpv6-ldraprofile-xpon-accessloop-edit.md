This macro configures xpon-access-loop-characteristics to DHCPv6 Relay Agent profile.

Input parameters:

* dhcpv6-profile-name: Name of the DHCPv6 Relay Agent profile
* vendor-specific-information: It adds access loop characteristics code 17 to the relayed RELAY FORW DHCPv6 messages
* xpon-access-loop-syntax: Access Loop sub-options that are to be added in option 17
