This macro detaches the DHCPv4 Relay Agent profile from VLAN sub-interface

Prerequisite:

* DHCPv4 Relay Agent profile must be attached to VLAN sub-interface.

Input parameters:

* cc-name: Ibridge user port name needs to be detached from VLAN sub interface
* ibridge-user-port-name: CC user port name needs to be detached from VLAN sub interface
