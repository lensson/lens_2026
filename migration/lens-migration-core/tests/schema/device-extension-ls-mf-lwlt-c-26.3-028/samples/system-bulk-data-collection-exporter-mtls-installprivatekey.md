This macro installs the private key to be used for a specific IPFIX Collector.  Please note that the client certificate is configured per collector.

Pre-condition:
The Client Certificate of specific IPFIX collector should be already configured via system-bulk-data-collection-exporter-tls-client-certificates-edit.

Input parameters:

* collector-name: Collector name
* exporter-name: Exporter name
* private-key: Private key value to be used
