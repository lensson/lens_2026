Description:
This macro attaches MoCA performance monitoring tca profiles to an ONT UNI interface (MoCA UNI)

Prerequisite:

* MoCA UNI interface to be created and PM enabled.
* MoCA Ethernet frame TCA profile should be created to associate it with tca-moca-ethernet-frame-profile
* MoCA Interface TCA profile should be be created to associate it with tca-moca-interface-profile

Input parameters:

* tca-moca-ethernet-frame-profile : moca-ethernet-frame-tca-profile name
* tca-moca-interface-profile : moca-interface-tca-profile name
Input parameters:

* moca-name: ONT UNI interface name (MoCA UNI)
* onu-name: ONU name
* tca-moca-ethernet-frame-profile: Moca Ethernet frame TCA profile.
* tca-moca-interface-profile: Moca Interface TCA profile.
