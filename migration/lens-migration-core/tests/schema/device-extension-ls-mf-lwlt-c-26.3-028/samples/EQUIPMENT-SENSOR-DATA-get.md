This macro retrieves operational data of the sensors in the device.

* Sensor data nodes present for any component of type sensor.
* Value and status of the sensor: e.g. temperature value,voltage,fanspeed.
* Sensor data provides information about sensors value-type, scale, precision & units.
* Sensor data also provides last obtained sensor value time and date (time & rate).

Pre-condition:

* Operational data for a component is only retrieved if the component is detected by the system as physically present and its parent component its oper-state is enabled.

Procedure:

* Retrieve operational data of the sesnors in the device.
