This is used to trigger a full data export of an on-change template. For a full-data-sync of a on-change template to be successful, connection with collector should be established and the on-change template should be enabled. This action will be used when the collector detects a mismatch in the cache-sequence-number in the exported data record of the on-change template, to update the inventory with the full data corresponding to the cache.

Input parameters:

* cache-name: cache name
