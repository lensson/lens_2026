This macro enables or disables the PM counters for an ENET interface on an ONU.

Prerequisite:

*   device has to be created.
*   ENET interface has to be created.

Input parameters:

* onu-name: ONU name
* onu-template-enet-name: ENET interface name
* pm-flag: **true** to enable or **false** to disable PM counters for an ENET
