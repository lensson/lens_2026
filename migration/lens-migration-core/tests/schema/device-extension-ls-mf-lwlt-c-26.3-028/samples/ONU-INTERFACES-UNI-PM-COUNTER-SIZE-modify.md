This macro modifies pm counter size of an UNI interface on an ONU.

Prerequisite:

*   device has to be created.
*   UNI interface has to be created.


Input parameters:

* onu-name: ONU name
* onu-template-uni-intf: ENET interface name
* pm_counter_size: Modifies pm counter size of an UNI interface,Values Range:[32bit-performance-monitoring,64bit-performance-monitoring]
