This macro deletes a QoS pbit scheduling policy along with several classifiers

* several classifiers are deleted based on operator's input.
* possible classifiers to be deleted include:
  * cls0-x
  * cls1-x
  * cls2-x
  * cls3-x
  * cls4-x
  * cls5-x
  * cls6-x
  * cls7-x

Pre-condition:

* OLT device has to be created.

Input parameters:

* pbit-scheduling-policy-name: Policy name
