Description:
This macro deletes the LLDP-MED parameters for the Ethernet UNI LLDP port and sets to default values in ONU.

Prerequisite:
- Ethernet UNI interface has to be created.
- LLDP-MED port parameters configured in Ethernet UNI port.
Input parameters:

* dest-mac-address: Ethernet destination mac-address.
* onu-name: ONU name
* onu-uni-intf: Ethernet UNI interface where LLDP-MED port is configured.
