Description:
This sample modifies the QoS policy profile.

Prerequisite:
device has to be created.

Input parameters:

* onu-name: ONU name
* qos-policy-name-1: First QoS Policy name
* qos-policy-name-2: Second QoS Policy name is empty if not specified
* qos-policy-name-3: Third QoS Policy name is empty if not specified
* qos-policy-profile-name: QoS Policy profile name
