Description:
This macro removes TCA profile for POTS call control performance monitoring counters in an ONU

Prerequisite:

* POTS call control PM TCA profile has to be created

Input parameters:

* ccpm-failure-tca-profile    :  ccpm-failure-tca-profile name

Input parameters:

* ccpm-failure-tca-profile: POTS Call Control performance monitoring TCA profile name
* onu-name: ONU name
