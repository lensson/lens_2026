This macro retrieves the pbit statistics on the VLAN sub-interface.

Output:

* discontinuity-time
* in-fwd-bytes
* in-fwd-frames
* in-discard-bytes
* in-discard-frames
* out-fwd-bytes
* out-fwd-frames
* out-discard-bytes
* out-discard-frames


Input parameters:

* vsi-interface-name: The name of the VLAN sub-interface on which the pbit statistics are enabled.
