This macro detaches the PPPoE Intermediate Agent profile with vlan range from an VALN sub-interface.

Prerequisite:

* PPPoE vlan-range-profile must be attached to VLAN sub-interface.

Input parameters:

* cc-name: The name of CC sample
* ibridge-user-port-name: Ibridge user port name
* vlanrange-profile-name: Vlan range profile name reference to protocol
