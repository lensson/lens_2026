
* This macro creates/modifies a multicast VPN profile.

Prerequisites:

* No prerequisites.

Input parameters:

* immediate-leave: Immediate leave controls the function by which an interface that receives a leave group message for a certain group might immediately delete the corresponding group record.
* multicast-protocol-version-v2: There is support for IGMP v2 (for IPv4) and/or MLD v1 (for IPv6).
* multicast-protocol-version-v3: There is support for IGMP v3 (for IPv4) and/or MLD v2 (for IPv6).
* vpn-profile-name: The name of a multicast proxy profile.
