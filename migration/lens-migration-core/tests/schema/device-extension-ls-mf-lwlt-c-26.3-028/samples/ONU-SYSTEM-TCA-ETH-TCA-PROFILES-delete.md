Description:
This macro removes the TCA profile for Ethernet frame performance monitoring counters in an ONU UNI interface (ENET)"

Prerequisite:
* Ethernet frame error TCA profile has to be created

Input parameters:
* ethernet-frame-error-profile    :  ethernet-frame-error-profile name

Input parameters:

* ethernet-frame-error-profile: Ethernet frame performance monitoring TCA profile name
* onu-name: ONU name
