This macro retrieves operational data of the temperature threshold values of each temperature sensor component.

* Displays temperature thresholds values of each temperature sesnor component: e.g. threshold-high,threshold-high-warning,threshold-high-clear.

Pre-condition:

* Operational data for a component is only retrieved if the component is detected by the system as physically present and its parent component its oper-state is enabled.

Procedure:

* Retrieve operational data of the temperature threshold values of each temperature sensor component.
