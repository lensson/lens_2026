
* Detach multicast package form multicast host interface

Prerequisites:

* The multicast vpn must exist.
* The multicast host interface must exist.
* The multicast package must exist.

Input parameters:

* multicast-package-name: The name of a multicast package.
* vpn-host-interface-name: The name of a multicast host interface.
* vpn-name: The name of a multicast vpn.
