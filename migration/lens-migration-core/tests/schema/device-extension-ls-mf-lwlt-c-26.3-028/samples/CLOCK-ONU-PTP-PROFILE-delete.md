This macro deletes the clock 8275dot1/ccsa ptp profile configuration data . The profile-name should be the exist ptp profile name in the equipment.

* every onu owns its own ptp profile

Input parameters:

* onu-name: the custom name of the onu.
* profile-name: the custom name of the ptp profile.
* profile-type: the spec type of the ptp profile. allowed values are [g8275dot1, ccsa]
