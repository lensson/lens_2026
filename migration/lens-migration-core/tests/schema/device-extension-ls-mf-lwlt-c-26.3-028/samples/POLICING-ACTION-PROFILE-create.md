This macro creates a policing-action-profile.

* Create a policing action profile including actions for different color packets
Input parameters:

* green-bac-color: bac color for green packets
* green-discard: true if packet is discard
* green-i-marking-dei: marking inner TAG dei for green packets
* green-i-marking-pbit: marking inner TAG p-bit for green packets
* green-metered-color: true if color is set by policer
* green-metered-flow: true if flow is metered by policer
* green-o-marking-dei: marking outer TAG dei for green packets
* green-o-marking-pbit: marking outer TAG p-bit for green packets
* policing-action-profile-name: policing action profile name
* red-bac-color: bac color for red packets
* red-discard: true if packet is discard
* red-i-marking-dei: marking inner TAG dei for red packets
* red-i-marking-pbit: marking inner TAG p-bit for red packets
* red-metered-color: true if color is set by policer
* red-metered-flow: true if flow is metered by policer
* red-o-marking-dei: marking outer TAG dei for red packets
* red-o-marking-pbit: marking outer TAG p-bit for red packets
* yellow-bac-color: bac color for yellow packets
* yellow-discard: true if packet is discard
* yellow-i-marking-dei: marking inner TAG dei for yellow packets
* yellow-i-marking-pbit: marking inner TAG p-bit for yellow packets
* yellow-metered-color: true if color is set by policer
* yellow-metered-flow: true if flow is metered by policer
* yellow-o-marking-dei: marking outer TAG dei for yellow packets
* yellow-o-marking-pbit: marking outer TAG p-bit for yellow packets
