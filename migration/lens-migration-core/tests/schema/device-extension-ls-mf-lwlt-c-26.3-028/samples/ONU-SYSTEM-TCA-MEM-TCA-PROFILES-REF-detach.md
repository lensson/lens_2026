Description:
This macro removes the TCA profile for Memory utilization on an ONU from a CPU component

Prerequisite:
 Memory tca profile has to be attached

Input parameters:
memory-tca-profile    :  memory-tca-profile name

Input parameters:

* cpu-component-name: CPU component name
* memory-tca-profile: memory-tca-profile name
* onu-name: ONU name
