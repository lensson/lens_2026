Description:
This sample retrieves the PM counters for a GEM port in an ONU.

Prerequisite:
Prerequisite:
*   device has to be created.
*   POTS interface of type voiceFXS created. 
*   POTS interworking node created.
*   PM is enabled for POTS interworking node.

Input parameters:

* gemport-name: GEM port name for which performance counters need to be fetched
* onu-name: ONU name
