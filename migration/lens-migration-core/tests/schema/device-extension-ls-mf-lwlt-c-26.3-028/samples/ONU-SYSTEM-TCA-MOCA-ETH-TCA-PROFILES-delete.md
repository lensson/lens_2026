Description:
This macro removes the TCA profiles for Moca Ethernet Performance Monitoring counter on an ONT MoCA UNI interface

Prerequisite:
* Moca ethernet frame TCA profile has to be created

Input parameters:
* moca-ethernet-frame-tca-profile : moca-ethernet-frame-tca-profile name

Input parameters:

* moca-ethernet-tca-profile: Moca Ethernet performance monitoring counters threshold profile name
* onu-name: ONU name
