This macro detaches ber-tca-profile of sd-threshold and sf-threshold from ANI Interface.

Pre-condition:

 * ber-tca-profile has to be created and associated to ANI container

Input parameters:

* ber-tca-profile: ber-tca-profile name


Input parameters:

* ani-name: The name on an Access Node Interface.
* ber-tca-profile: ber-tca-profile name
* onu-name: ONU name
