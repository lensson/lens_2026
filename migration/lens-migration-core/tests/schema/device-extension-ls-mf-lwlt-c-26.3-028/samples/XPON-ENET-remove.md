This macro deletes an ENET port.

Input parameters:

* enet-name: The name of the virtual ENET UNI.
* onu-name: ONU name
* onu-template-enet-name: ENET interface name
* tc0-gemport-set-name: GEMPORT set name to indicate tcont no-sharing, uni-sharing or onu-sharing for traffic class 0
* tc1-gemport-set-name: GEMPORT set name to indicate tcont no-sharing, uni-sharing or onu-sharing for traffic class 1
* tc2-gemport-set-name: GEMPORT set name to indicate tcont no-sharing, uni-sharing or onu-sharing for traffic class 2
* tc3-gemport-set-name: GEMPORT set name to indicate tcont no-sharing, uni-sharing or onu-sharing for traffic class 3
* tc4-gemport-set-name: GEMPORT set name to indicate tcont no-sharing, uni-sharing or onu-sharing for traffic class 4
* tc5-gemport-set-name: GEMPORT set name to indicate tcont no-sharing, uni-sharing or onu-sharing for traffic class 5
* tc6-gemport-set-name: GEMPORT set name to indicate tcont no-sharing, uni-sharing or onu-sharing for traffic class 6
* tc7-gemport-set-name: GEMPORT set name to indicate tcont no-sharing, uni-sharing or onu-sharing for traffic class 7
