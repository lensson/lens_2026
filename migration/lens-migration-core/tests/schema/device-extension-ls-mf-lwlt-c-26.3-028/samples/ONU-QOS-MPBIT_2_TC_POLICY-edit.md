Description:
This sample configures the marked pbit to traffic class parameters.

Prerequisite:
device has to be created.
Ethernet port has to be created

Input parameters:

* mpbit-2-tc-policy: Policy name
* mpbit0-tc: Value for pbit marking list to traffic class 0
* mpbit1-tc: Value for pbit marking list to traffic class 1
* mpbit2-tc: Value for pbit marking list to traffic class 2
* mpbit3-tc: Value for pbit marking list to traffic class 3
* mpbit4-tc: Value for pbit marking list to traffic class 4 (if specified)
* mpbit5-tc: Value for pbit marking list to traffic class 5 (if specified)
* mpbit6-tc: Value for pbit marking list to traffic class 6 (if specified)
* mpbit7-tc: Value for pbit marking list to traffic class 7 (if specified)
* onu-name: ONU name
