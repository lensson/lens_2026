This macro deletes a QoS Queue color policy along with several classifiers

* several classifiers are deleted based on operator's input.
* possible classifiers to be created include:
  * classifier0-x
  * classifier1-x
  * classifier2-x
  * classifier3-x
  * classifier4-x
  * classifier5-x
  * classifier6-x
  * classifier7-x

Pre-condition:

* OLT device has to be created.

Input parameters:

* color-policy-name: Policy name
