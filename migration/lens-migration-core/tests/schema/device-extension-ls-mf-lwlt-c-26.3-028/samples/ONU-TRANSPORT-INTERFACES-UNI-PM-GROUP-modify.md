Description:
This macro modifies pm group of an UNI interface on an ONU.

Prerequisite:
*   device has to be created.
*   UNI interface has to be created.

Input parameters:

* eth-port-hw-option: ONU Ethernet HW Option,Values Range:[ethernetCsmacd]
* onu-name: ONU name
* onu_uni_intf: UNI interface name
* pm_group: Modifies pm group of an UNI interface,Values :[ ethernet-us-pm ethernet-ds-pm ethernet-phy-pm ]
