Description:
This macro removes the ANI XGPON downstream performance monitoring TCA profile for an ONT ANI interface

Prerequisite:

* ANI XGPON downstream PM TCA profile has to be created

Input parameters:

* xgpon-ds-layer-tca-profile    :  xgpon-ds-layer-tca-profile name

Input parameters:

* onu-name: ONU name
* xgponani-downstream-error-profile: ANI XGPON downstream performance monitoring counters threshold profile name
