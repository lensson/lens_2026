Description:
This macro detaches Call control and RTP TCA profiles for POTS interworking performance monitoring counters from an ONU

Prerequisite:

* POTS Call control PM TCA profile should be already associated with ccpm-failure-tca-profile
* POTS RTP PM TCA profile should be already associated with rtp-failure-tca-profile

Input parameters:

* ccpm-failure-tca-profile    :  ccpm-failure-tca-profile name
* rtp-failure-tca-profile    :  rtp-failure-tca-profile name

Input parameters:

* ccpm-failure-tca-profile: POTS Call control PM TCA profile name
* onu-name: ONU name
* pots-name: POTS interworking node for which for which TCA should be enabled or disabled
* rtp-failure-tca-profile: POTS RTP PM TCA profile name
