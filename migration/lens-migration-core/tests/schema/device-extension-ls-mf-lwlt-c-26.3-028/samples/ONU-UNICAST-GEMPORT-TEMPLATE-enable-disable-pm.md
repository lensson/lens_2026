Description:
This sample enables or disables the PM counters for a GEM port in an ONU.

Prerequisite:
*   device has to be created.
*   ONU ANI has to be created.
*   GEM port has to be created.
Input parameters:

* onu-name: ONU name
* onu-template: ONU template name
* onu-template-gemport-name: GEM port name for which performance to be enabled or disabled
* pm-flag: **true** to enable or **false** to disable PM counters for a GEM port
