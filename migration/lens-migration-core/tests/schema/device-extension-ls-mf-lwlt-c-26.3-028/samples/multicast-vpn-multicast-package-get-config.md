
* This macro retrieves the multicast package configuration from the VPN.

Prerequisite:

* The multicast vpn must exist.

Input parameters:

* detailed-output: Provides detailed configuration of all the multicast packages in the VPN in the system when enabled and a list of configured multicast package names in the VPN when disabled. This argument will be ignored when the multicast-package-name is provided.
* multicast-package-name: The name of the multicast package.
* vpn-name: The name of the multicast VPN.
