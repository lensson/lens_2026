This macro creates transceiver on the board.

Input parameters:

* admin-state: Transceiver's administrative permission to provide service
* cage-name: The name of cage
* power-saving: Power saving configuration at transceiver level. When no power-saving criteria configured, it will inherit the system level power-saving criteria.
* transceiver-model-name: The generic or 3FE specific model name of transceiver
* transceiver-name: The name of transceiver
