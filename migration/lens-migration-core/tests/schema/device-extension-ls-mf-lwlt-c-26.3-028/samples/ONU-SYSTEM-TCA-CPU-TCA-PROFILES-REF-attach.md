Description:
This macro attaches cpu-tca-profile for the CPU utilization threshold to CPU component.

Prerequisite:
 CPU TCA profile has to be created.

Input parameters:
cpu-tca-profile    :  cpu-tca-profile name

Input parameters:

* cpu-component-name: CPU component name
* cpu-tca-profile: cpu-tca-profile name
* onu-name: ONU name
