Description:
This macro removes TCA profile for POTS RTP performance monitoring counters in an ONU

Prerequisite:

* POTS RTP PM TCA profile has to be created

Input parameters:

* rtp-failure-tca-profile    :  rtp-failure-tca-profile name

Input parameters:

* onu-name: ONU name
* rtp-failure-tca-profile: POTS RTP performance monitoring TCA profile name
