Description:
This macro removes the TCA profile for Ethernet physical layer performance counters in an ONU UNI interface (ENET)

Prerequisite:

* Ethernet physical layer error TCA profile has to be created

Input parameters:

* ethernet-physical-layer-error-profile    :  ethernet-physical-layer-error-profile name

Input parameters:

* ethernet-physical-layer-error-profile: Ethernet physical layer performance monitoring threshold profile name
* onu-name: ONU name
