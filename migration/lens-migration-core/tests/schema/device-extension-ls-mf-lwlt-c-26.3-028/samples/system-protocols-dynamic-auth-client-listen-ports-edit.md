This macro configures listen-ports of Dynamic Authorization Server to handle dynamic authorization requests

Input parameters:

* listen-port1: The Dynamic Authorization Server (DAS) listens on UDP port 3799 to handle dynamic authorization requests from the Dynamic Authorization Client (DAC).
* listen-port2: The Dynamic Authorization Server (DAS) listens on UDP port 1700 to handle dynamic authorization requests from the Dynamic Authorization Client (DAC).
* listen-port3: The Dynamic Authorization Server (DAS) listens on UDP port 1813 to handle dynamic authorization requests from the Dynamic Authorization Client (DAC).
