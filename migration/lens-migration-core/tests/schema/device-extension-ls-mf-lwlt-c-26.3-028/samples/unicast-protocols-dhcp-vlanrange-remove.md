This macro detaches an DHCPv4 Relay Agent profile with vlan range profile from an VLAN sub-interface.

Prerequisite:

* DHCPv4 vlan-range-profile must be attached to VLAN sub-interface

Input parameters:

* cc-name: The name of CC sample
* ibridge-user-port-name: Ibridge user port name
* vlanrange-profile-name: Vlan range profile name reference to protocol
