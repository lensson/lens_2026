Description:
This macro creates pm group of a MoCA UNI interface on an ONU.

Prerequisite:
*   device has to be created.
*   UNI interface has to be created.

Input parameters:

* moca-name: MoCA uni interface name
* onu-name: ONU name
* pm_group: Create pm group of a MoCA UNI interface,Values :[ moca-interface-pm moca-ethernet-pm ]
