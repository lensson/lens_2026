Description:
This sample configures TR069 server to onu.

Prerequisite:
ONU device has to be created.

Input parameters:

* ACS-server-uri: ACS server uri
* admin-state: In almost all the deployment scenarios, the TR-069 connection is configured with unlocked.
* authentication-user-name: Authentication user name
* authentication-user-password: authentication user password.
* authentication-validation-scheme: In almost all the deployment scenarios, validation scheme with md5-digest is selected
* management-dei: TR-069 management traffic dei
* management-pbit: TR-069 management traffic pbit
* management-VLAN: TR-069 management traffic VLAN
* onu-name: ONU name
* reference-onu-v-vrefpoint: reference onu vrefpoint interface name
