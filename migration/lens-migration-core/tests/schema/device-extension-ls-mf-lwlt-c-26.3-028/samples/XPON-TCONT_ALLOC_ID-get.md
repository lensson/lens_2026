This macro generates the TCONT ALLOC ID parameters.

Prerequisite:

*The referenced Channel group must exist in OLT side.

Input parameters:

* allowed-alloc-id-list: An allowed ID list for allocating free ALLOC IDs, the format can be 257-1023 or 256,270,1023 or 256-900,1000-1023
* olt-port-name: Channel group
* tcont-name: TCONT Interface name
* xpon-technology: Type of Pon
