This macro retrieves the operational data of IPv4 antispoofing on the VLAN sub-interface.

Prerequisite:

* ipv4-security-statistics must be enabled on the VLAN sub-interface.

Input parameters:

* cc-name: The name of CC sample. In IBRIDGE senario, leave it empty
* ibridge-user-port-name: The name of IBRIDGE-USER-PORT sample. In CC senario, leave it empty
