This macro gets configuration of a BER TCA profile. It contains two thresholds: a signal degrade (SD) and signal failure (SF) threshold. The specified value x represents the negative power of ten in 10^-x. The range of x for the SD threshold is [4-10] and 9 is the default value. The range for the SF threshold is [3-8] with a default value of 5.

Input parameters:

* ber-tca-profile: Name of the ber-tca profile.
