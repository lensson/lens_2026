This macro configures a Traffic Class to Queue mapping profile.

Prerequisite:

* No prerequisite.


Input parameters:

* queue0: Local Queue-Id 0
* queue0-tc: Traffic Class to Queue0
* queue1: Local Queue-Id 1
* queue1-tc: Traffic Class to Queue1
* queue2: Local Queue-Id 2
* queue2-tc: Traffic Class to Queue2
* queue3: Local Queue-Id 3
* queue3-tc: Traffic Class to Queue3
* queue4: Local Queue-Id 4
* queue4-tc: Traffic Class to Queue4
* queue5: Local Queue-Id 5
* queue5-tc: Traffic Class to Queue5
* queue6: Local Queue-Id 6
* queue6-tc: Traffic Class to Queue6
* queue7: Local Queue-Id 7
* queue7-tc: Traffic Class to Queue7
* tc2q-profile-name: Name of TC to Queue Id Mapping Profile
