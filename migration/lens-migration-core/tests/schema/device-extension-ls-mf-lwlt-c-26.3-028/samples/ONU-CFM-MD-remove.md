To delete Connectivity Fault Management's (CFM) MD in ONU this macro configures following data nodes:

- Configuring the **md-id** identified by a string that defines the index to the Maintenance Domain list which needs to be deleted

Prerequisite:
MD has to be created. MEP must be removed from MG.

Input parameters:

* md-id: The ID of MD that defines the index to the Maintenance Domain list which needs to be deleted
* onu-name: ONU name
