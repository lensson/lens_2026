Description:
This macro creates pm group of POTS interworking node on an ONU.

Prerequisite:
*   device has to be created.
*   POTS interface of type voiceFXS created.
*   POTS interworking node created.

Input parameters:

* onu-name: ONU name
* pm_group: Creates pm group of an POTS interworking node,Values :[ call-control-pm rtp-pm ]
* pots-name: POTS interworking node for which performance group
