This macro configures the EST certificate and CA for callhome.

Pre-condition: The pinned certificates list or the EST profile(s) must be configured.
Input parameters:

* client-auth-est-profile-name: The EST CA profile name to be used for Mutual TLS with the CallHome server
* netconf-client-name: Name of NETCONF clients list
* server-auth-est-profile-name: The EST profile name to be used for Mutual TLS with the CallHome server
