
* This macro removes a multicast channel.

Prerequisites:

* The multicast VPN must exist.
* The multicast channel must exist.
* Package name must be provided if the channel is associated to any package.

Input parameters:

* multicast-channel-name: The name of a multicast channel.
* multicast-package-name: The name of a multicast package.
* vpn-name: The name of a multicast vpn.
