Description:
This macro removes the ANI XGPON TC layer performance monitoring TCA profile for an ONT ANI interface

Prerequisite:

* ANI XGPON TC layer PM TCA profile has to be created

Input parameters:

* xgpon-tc-layer-tca-profile    :  xgpon-tc-layer-tca-profile name

Input parameters:

* onu-name: ONU name
* xgpon-tc-layer-tca-profile: ANI XGPON TC layer PM counters threshold profile name
