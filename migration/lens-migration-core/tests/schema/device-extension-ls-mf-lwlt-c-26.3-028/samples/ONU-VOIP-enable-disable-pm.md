Description:
This sample enables or disables the PM counters for a POTS interworking node.

Prerequisite:
*   device has to be created.
*   POTS interface of type voiceFXS created. 
*   POTS interworking node created.
Input parameters:

* onu-name: ONU name
* pm-flag: **true** to enable or **false** to disable PM counters for POTS interworking node
* pots-name: POTS interworking node for which performance to be enabled or disabled
