This macro retrieves DHCPv4 Relay Agent statistics.

Prerequisite:

* DHCPv4 Relay Agent must be enabled on user VLAN sub-interface

Note : Reset of VLAN sub-interface statistics resets DHCPv4 Relay Agent statistics too

Input parameters:

* cc-name: The name of CC user port name for L2 DHCPv4 Relay Agent statistics counter needs to be retrieved
* ibridge-user-port-name: The name of Ibridge user port name for L2 DHCPv4 Relay Agent statistics counter needs to be retrieved
