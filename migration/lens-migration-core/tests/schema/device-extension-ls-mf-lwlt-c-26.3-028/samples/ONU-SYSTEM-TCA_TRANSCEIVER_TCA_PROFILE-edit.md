This macro configures SFP transceiver tca profile RSSI for temperature and voltage.

Pre-condition: 

 * SFP transceiver has to be created.

Input parameters:

* temperature-threshold-high-alarm: Value for temperature-threshold-high-alarm range is -45000 .. 90000
* temperature-threshold-high-warning: Value for temperature-threshold-high-warning range is -45000 .. 90000
* temperature-threshold-low-alarm: Value for temperature-threshold-low-alarm range is -45000 .. 90000
* temperature-threshold-low-warning: Value for temperature-threshold-low-warning range is -45000 .. 90000
* transceiver-tca-profile-name: transceiver-tca-profile-name
* voltage-threshold-high-alarm: voltage-threshold-high-alarm range is 0 .. 65500
* voltage-threshold-high-warning: Value for voltage-threshold-high-warning range is 0 .. 65500
* voltage-threshold-low-alarm: Value for voltage-threshold-low-alarm range is 0 .. 65500
* voltage-threshold-low-warning: Value for voltage-threshold-low-warning range is 0 .. 65500


Input parameters:

* onu-name: ONU name
* temperature-threshold-high-alarm: Value for TEMPERATURE-THRESHOLD-HIGH-ALARM range is -45000 .. 90000
* temperature-threshold-high-warning: Value for TEMPERATURE-THRESHOLD-HIGH-WARNING range is -45000 .. 90000
* temperature-threshold-low-alarm: Value for TEMPERATURE-THRESHOLD-LOW-ALARM range is -45000 .. 90000
* temperature-threshold-low-warning: Value for TEMPERATURE-THRESHOLD-LOW-WARNING range is -45000 .. 90000
* transceiver-tca-profile-name: TRANSCEIVER-TCA-PROFILE-NAME
* voltage-threshold-high-alarm: VOLTAGE-THRESHOLD-HIGH-ALARM range is 0 .. 65500
* voltage-threshold-high-warning: Value for VOLTAGE-THRESHOLD-HIGH-WARNING range is 0 .. 65500
* voltage-threshold-low-alarm: Value for VOLTAGE-THRESHOLD-LOW-ALARM range is 0 .. 65500
* voltage-threshold-low-warning: Value for VOLTAGE-THRESHOLD-LOW-WARNING range is 0 .. 65500
