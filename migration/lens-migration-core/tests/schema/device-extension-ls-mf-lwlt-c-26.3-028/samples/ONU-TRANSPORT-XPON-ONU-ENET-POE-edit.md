Description:
This sample configures the power over ethernet control for ONU Ethernet port.

Prerequisite:
device has to be created.

Input parameters:

* eth-port-hw-name: name of ONU ENET port
* onu-name: ONU name
* power-over-ethernet-control: This attribute is defined on port, it is used to config power-over-ethernet-control,only support for rj45*
* power-priority: Indicates that the most highst power priority
* pse-class-control: Used to place specific limits on the class of power supported by this port.
