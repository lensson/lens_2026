The macro works for the enable or disable the ssmout function of LT frequency interface.

Input parameters:

* lt-interface-name: the interface name of the LT board.
* ssm-out-enable: the ssmout status of the LT frequency interface(true/false).
* ssm-out-vlan-id: the ID of LT SSM Out Vlan. It should use the same ESMC-VLAN ID with ONU.
* vlan-profile-name: the name of LT SSM Out profile.
