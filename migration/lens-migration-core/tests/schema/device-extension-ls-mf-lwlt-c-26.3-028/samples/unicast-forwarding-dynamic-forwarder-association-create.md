This macro enables a VLAN sub-interface for l2 forwarder dynamic association through 802.1x Change Of Authorisation.

Pre-condition:

* VLAN sub-interface has to be configured with interface-usage type as 'user-port'.
~


Input parameters:

* l2f-association-control: The identity reference to the association-control in l2-forwarder-dynamic-association container.
* vsi-interface-name: The name of the VLAN sub-interface to configure l2-forwarder-dynamic-association.
