
* This macro retrieves the multicast channel configuration from the VPN.

Prerequisite:

* The multicast VPN must exist.

Input parameters:

* detailed-output: Provides detailed configuration of all the multicast channels in the VPN in the system when enabled and a list of configured multicast channel names in the VPN when disabled. This argument will be ignored when the multicast-channel-name is provided.
* multicast-channel-name: The name of the multicast channel.
* vpn-name: The name of the multicast VPN.
