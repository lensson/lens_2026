Description:
This macro attaches TCA profile for GEM port performance monitoring counters in an ONU

Prerequisite:

* GEM port to be created and PM enabled.
* GEM port TCA profile should be created to associate it with gemport-tca-profile

Input parameters:

* gemport-tca-profile    :  gemport-tca-profile name

Input parameters:

* gemport-name: GEM port name for which TCA should be enabled or disabled
* gemport-tca-profile: GEM port PM TCA profile name
* onu-name: ONU name
