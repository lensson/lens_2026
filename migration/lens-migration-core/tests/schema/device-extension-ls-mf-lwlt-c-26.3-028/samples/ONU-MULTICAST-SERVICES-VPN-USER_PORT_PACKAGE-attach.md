Attach multicast package to multicast user port


Pre-condition:

* The multicast vpn named VPN-NAME has to be created.
* The multicast user port named USER-PORT-NAME has to be created.
* The multicast package named MULTICAST-PACKAGE-NAME has to be created.




Description:

* Attach a multicast package to a multicast user port.

Input parameters:

* multicast-package-name: The name of a multicast package.
* onu-name: ONU name
* user-port-name: The name of a multicast user port.
* vpn-name: The name of a multicast vpn.
