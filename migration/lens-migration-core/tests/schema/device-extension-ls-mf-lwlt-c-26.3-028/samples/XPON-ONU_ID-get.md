This macro generates the ONU ID parameters.

Prerequisite:

*The referenced Channel group must exist in OLT side.

Input parameters:

* allowed-onu-id-list: An allowed ID list for allocating free ONU IDs, the format can be 1-127 or 1,10,120 or 1-30,90-127
* olt-port-name: Channel group
* onu-name: the ONU name
* xpon-technology: Type of Pon
