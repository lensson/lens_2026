This macro enables interface statistics.

Pre-condition:

* VANI-INTERFACE-NAME

* There are best-effort/false 2 modes.
*	best-effort: This is the default option, under this, in/out pkts, in/out discards, in/out octets and in/out dropped-bytes are supported and collected.
*	false: under this option, statistics are not collected.

Input parameters:

* interface-statistics-mode: 
* vani-interface-name: The name of the v-ani interface who will enable the statistics.
