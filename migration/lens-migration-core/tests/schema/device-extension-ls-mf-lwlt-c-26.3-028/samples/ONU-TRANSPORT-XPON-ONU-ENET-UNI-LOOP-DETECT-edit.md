Description:
This sample enables or disables the UNI loop detect for an ONU.

Prerequisite:
ONU hardware component chassis has to be created.

Input parameters:

* auto-shut-off: **true** to do auto shut off or **false** to not auto shut off for an ONT
* enable-flag: **true** to enable or **false** to disable loop detect for an ONT
* onu-name: ONU name
