Description:
This macro configures TCA profiles for the Memory utilization of ONU CPU component.

Prerequisite:
 CPU component has to be created.

Input parameters:
memory-threshold :  Value for Memory threshold range is 0..100

Input parameters:

* memory-tca-profile: Memory threshold profile name
* memory-tca-threshold: value for threshold range is 0..100
* onu-name: ONU name
