This macro retrieves the configuration data of Traffic class to Queue mapping profile.

Prerequisite: 

 No prerequisite.

Input parameters:

* detailed-output: Provides detailed profile configuration data of all the TC-2-Queue mapping profiles in the system when enabled and only TC-2-Queue mapping profile names when disabled. This argument will be ignored when tc2q-prof-name is provided.
* tc2q-prof-name: Name of the TC-2-Queue mapping profile. If this is not provided, then all the configured TC-2-Queue mapping profiles will be retrieved.
