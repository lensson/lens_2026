Description:
This macro attaches Call control and RTP TCA profiles for POTS interworking performance monitoring counters in an ONU

Prerequisite:

* POTS interworking node created and PM enabled.
* POTS interworking Call control PM TCA profile should be created to associate it with ccpm-failure-tca-profile
* POTS interworking RTP PM TCA profile should be created to associate it with rtp-failure-tca-profile

Input parameters:

* ccpm-failure-tca-profile    :  ccpm-failure-tca-profile name
* rtp-failure-tca-profile    :  rtp-failure-tca-profile name

Input parameters:

* ccpm-failure-tca-profile: POTS Call control PM TCA profile name
* onu-name: ONU name
* pots-name: POTS interworking node for which for which TCA should be enabled or disabled
* rtp-failure-tca-profile: POTS RTP PM TCA profile name
