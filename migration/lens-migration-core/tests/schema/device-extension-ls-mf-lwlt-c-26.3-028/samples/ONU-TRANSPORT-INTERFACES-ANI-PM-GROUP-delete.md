Description:
This macro deletes pm group of an ANI interface on an ONU.
Prerequisite:
*   device has to be created.
*   ANI interface has to be created.

Input parameters:

* onu-name: ONU name
* onu_ani_intf: ANI interface name
* pm_group: Deletes pm group of an ANI interface,Values :[ fec-pm bip-pm xgpon-tc-pm xgpon-ds-pm xgpon-us-pm aggr-gem-pm ]
