This macro displays the ethernet speed for a given Uplink interface.

Pre-condition:

* SFP should be plugged-in on the cage
* Configure SFP in Copper platforms using macro "equipment-sfp-edit"
* Configure interface in Copper platforms using macro "transport-ethernet-interface-edit"
* Configure interface in Fiber platforms using macro "transport-ethernet-interface-uplink-edit"


Input parameters:

* network-port-name: Name of interface
