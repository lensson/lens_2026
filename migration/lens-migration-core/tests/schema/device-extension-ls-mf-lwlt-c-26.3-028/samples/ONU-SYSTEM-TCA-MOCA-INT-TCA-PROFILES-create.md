Description:
This macro configures TCA profiles for Moca Interface Performance Monitoring counter on an ONT MoCA UNI interface.

Prerequisite:
* None

Input parameters:
* in-errored-threshold: Threshold value for the incoming errored packets where range is 1 .. 4294967294
* in-errored-and-missed-threshold: Threshold value for the incoming errored and missed packets where range is 1 .. 4294967294

Input parameters:

* in-errored-and-missed-threshold: Threshold value for the incoming errored and missed packets where range is 1 .. 4294967294
* in-errored-threshold: Threshold value for the incoming errored packets where range is 1 .. 4294967294
* moca-interface-tca-profile: Moca Interface performance monitoring counters threshold profile name
* onu-name: ONU name
