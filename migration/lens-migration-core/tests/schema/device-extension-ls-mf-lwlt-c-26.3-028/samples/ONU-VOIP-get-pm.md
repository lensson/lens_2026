Description:
This sample retrieves the PM counters for a POTS interworking node in an ONU.

Prerequisite:
*   device has to be created.
*   POTS interface of type voiceFXS created. 
*   POTS interworking node created.
*   PM is enabled in the GEM port
Input parameters:

* onu-name: ONU name
* pots-name: POTS interworking node name for which performance counters need to be fetched
