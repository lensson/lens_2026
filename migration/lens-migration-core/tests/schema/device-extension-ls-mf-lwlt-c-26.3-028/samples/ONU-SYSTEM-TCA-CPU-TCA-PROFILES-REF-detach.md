Description:
This macro detaches cpu-tca-profile for the CPU utilization threshold from a CPU component.

Prerequisite:
 CPU TCA profile has to be attached.

Input parameters:
cpu-tca-profile    :  cpu-tca-profile name

Input parameters:

* cpu-component-name: CPU component name
* cpu-tca-profile: cpu-tca-profile name
* onu-name: ONU name
