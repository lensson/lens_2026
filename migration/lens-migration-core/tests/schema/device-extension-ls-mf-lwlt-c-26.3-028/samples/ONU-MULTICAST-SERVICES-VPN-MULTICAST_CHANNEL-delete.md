Delete multicast channel


Pre-condition:

* The multicast vpn named VPN-NAME has to be created.
* The multicast channel named MULTICAST-CHANNEL-NAME has to be created.




Description:

* Delete multicast channel for multicast vpn.

Input parameters:

* multicast-channel-name: The name of a multicast channel.
* multicast-package-name: The name of a multicast package.
* onu-name: ONU name
* vpn-name: The name of a multicast vpn.
