This macro config the embeded ONU release

Pre-condition:

* onu has to be create

Input parameters:

* active-software-release: ACTIVE-SOFTWARE-RELEASE-NAME
* onu-name: ONU name
* passive-software-release: PASSIVE-SOFTWARE-RELEASE-NAME
* target-active-vendor-specific-software: This is a string describing the active vendor-specific file package name to be applied on the ONU
* target-delayed-activation: TARGET-SOFTWARE-CONTROL
* target-passive-vendor-specific-software: This is a string describing the passive vendor-specific file package name to be applied on the ONU
* vendor-sprcific-target-delayed-activation: This is delay control for vendor-specific file, value true means delayed-activation.
* vendor-sprcific-type: The vendor specific SW Image pair
