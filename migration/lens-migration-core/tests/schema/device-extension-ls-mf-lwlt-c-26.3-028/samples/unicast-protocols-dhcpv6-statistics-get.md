This macro retrieves a DHCPv6 Agent Relay statistics.

Prerequisite:

* DHCPv6 Relay Agent must be enabled on VLAN sub-interface

Note : Reset of VLAN sub-interface statistics resets DHCPv6 Relay statistics too

Input parameters:

* cc-name: The name of CC sample
* ibridge-user-port-name: Ibridge user port name
