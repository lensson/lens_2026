This macro creates secondary inband management interface and its lower layers.

Input parameters:

* secondary-forwarder-name: Name of secondary forwarder
* secondary-inband-interface-name: Name of secondary inband interface
* secondary-l2term-interface-name: Name of secondary l2 termination interface
* secondary-vlan-subinterface-name: Name of secondary vlan sub-interface
* subif-lowerlayer: Subif lower layer
* tag-value: Tagging of secondary inband interface
* vlan-id: Vlan value
